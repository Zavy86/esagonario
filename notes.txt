to use shared files in nest:

 - update tsconfig.json adding the following lines
   "compilerOptions": {
      ...
        "paths": {
        "@shared/*": [ "../objects/*" ],
      }
   }

 - update nest-cli.json adding the following lines
   {
     ...
     "entryFile": "backend/src/main",
     ...
   }

 - update package.jon start:prod script to
   "start:prod": "node dist/backend/src/main",
